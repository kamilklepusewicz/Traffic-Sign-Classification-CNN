# Klasyfikacja Znaków Drogowych (GTSRB)

Projekt zaliczeniowy z przedmiotu Podstawy Sieci Neuronowych.

## Autorzy
- Kamil Klepusewicz
- Szymon Brodziak

## Opis
Projekt wykorzystuje sieci CNN do klasyfikacji znaków drogowych.
Osiągnięto skuteczność **99.77%**.

## Pliki
- `notebook.ipynb` - Kod źródłowy projektu (Python/Keras).
- `sprawozdanie.pdf` - Pełny raport z projektu.
- `prezentacja.pdf` - Slajdy podsumowujące.
